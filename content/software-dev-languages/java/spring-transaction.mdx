### 事务管理
- 编程式事务（TransactionTemplate）
- 声明式事务（@Transactional 注解）
    标注在类上：所有 public 且非静态方法均启用事务
    可以使用 osolation 声明事务的隔离级别
    使用 propagation 声明事务的传播机制
