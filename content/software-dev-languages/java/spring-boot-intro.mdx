# Spring Boot 微服务实战入门

Spring Boot 是 Java 生态中最流行的微服务框架，它极大地简化了 Spring 应用的开发流程。

## 快速开始

创建一个简单的 REST API：

```java
@RestController
@RequestMapping("/api/users")
public class UserController {

    @Autowired
    private UserService userService;

    @GetMapping("/{id}")
    public ResponseEntity<User> getUser(@PathVariable Long id) {
        return userService.findById(id)
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }

    @PostMapping
    public ResponseEntity<User> createUser(@RequestBody @Valid UserDTO dto) {
        User user = userService.create(dto);
        return ResponseEntity.status(HttpStatus.CREATED).body(user);
    }
}
```

## 配置管理

Spring Boot 使用 `application.yml` 进行配置管理：

```yaml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/mydb
    username: postgres
    password: ${DB_PASSWORD}
```

## 依赖注入

Spring Boot 的核心是 IoC 容器和依赖注入：

```java
@Service
public class UserService {
    private final UserRepository repository;

    public UserService(UserRepository repository) {
        this.repository = repository;
    }
}
```

## 总结

Spring Boot 提供了开箱即用的微服务开发体验，通过自动配置和约定优于配置的原则，让开发者可以专注于业务逻辑。
