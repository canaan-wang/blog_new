HTML 的全称是 `HyperText Markup Language（超文本标记语言）`，它不是编程语言（不能实现逻辑运算），而是 `标记语言`，用来定义网页的结构和内容。

`超文本`：指页面内可以包含链接、图片、音频、视频等非纯文本内容。

`标记语言`：通过一系列预定义的「标签」来描述内容的含义（比如"这是标题""这是段落""这是图片"）。

简单来说，HTML 就是网页的"骨架"——决定了网页里有什么、按什么结构排列，而后续的 CSS 负责"美化"（样式）、JavaScript 负责"交互"（动态效果）。

---

## 基本文档结构

一个完整的 HTML 文件有固定的骨架，所有内容都包裹在这个骨架里：

```html
<!DOCTYPE html>  <!-- 声明文档类型为HTML5，必须放在第一行 -->
<html lang="zh-CN">  <!-- 根标签，lang指定页面语言（zh-CN为中文） -->
<head>  <!-- 头部：存放页面元信息（不显示在页面上） -->
    <meta charset="UTF-8">  <!-- 字符编码，UTF-8支持所有语言 -->
    <title>我的第一个HTML页面</title>  <!-- 页面标题（显示在浏览器标签） -->
</head>
<body>  <!-- 主体：页面可见的所有内容都放在这里 -->
    <h1>欢迎学习HTML</h1>  <!-- 一级标题 -->
    <p>这是第一个段落内容</p>  <!-- 段落 -->
</body>
</html>
```

---

## 核心概念

`标签（Tag）`：HTML 的基本单位，通常成对出现（如 `<h1></h1>`），少数单标签（如 `<img />`）。

开始标签：`<标签名>`（如 `<p>`），表示内容开始。

结束标签：`</标签名>`（如 `</p>`），表示内容结束。

`属性（Attribute）`：附加在标签上的"参数"，用来修饰标签，格式为 `属性名="属性值"`，比如：

```html
<img src="logo.png" alt="网站logo">  <!-- src指定图片路径，alt是图片加载失败时的替代文本 -->
<a href="https://www.baidu.com">百度</a>  <!-- href指定链接地址 -->
```

`元素（Element）`：标签 + 标签内的内容，比如 `<p>这是段落</p>` 就是一个段落元素。

---

## 常用标签

| 标签类型 | 标签示例 | 作用 |
|----------|----------|------|
| 标题     | `<h1>`-`<h6>` | 定义标题，h1最大，h6最小 |
| 段落     | `<p>` | 定义文本段落 |
| 链接     | `<a href="网址">` | 定义超链接 |
| 图片     | `<img src="路径" alt="描述">` | 插入图片 |
| 列表     | `<ul>`（无序列表）/`<ol>`（有序列表）+`<li>` | 定义列表项 |
| 容器     | `<div>`（块级）/`<span>`（行内） | 划分页面区域，用于布局 |
| 表单     | `<input>`/`<button>`/`<select>` | 收集用户输入（如登录框、搜索框） |

---

## 版本与现状

目前主流使用的是 `HTML5`（2014年正式发布），相比旧版本：

1. 新增了语义化标签（如 `<header>` 头部、`<nav>` 导航、`<footer>` 底部），让页面结构更清晰。
2. 原生支持音频/视频（`<audio>`/`<video>`），无需依赖插件。
3. 支持本地存储、画布（`<canvas>`）等新特性，拓展了网页功能。

---

## 运行 HTML 文件

1. 用记事本、VS Code 等编辑器编写代码，保存为 `xxx.html`（后缀必须是 .html）。
2. 双击该文件，会自动用默认浏览器打开，就能看到页面效果。

---

## 总结

HTML 是网页的"骨架"，通过标签定义页面结构和内容，非编程语言。

核心构成包括标签、属性、元素，HTML5 是当前主流版本，新增了语义化、音视频等特性。

编写好的 HTML 文件可直接用浏览器打开运行，是前端开发的基础。

掌握 HTML 后，搭配 CSS（美化样式）和 JavaScript（实现交互），就能开发出完整的网页了。
