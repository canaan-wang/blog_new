Transformer 架构

Transformer 是现代大语言模型的基础架构，主要由编码器和解码器组成：

graph TD
    A[输入文本] --> B[词嵌入
Embedding]
    B --> C[位置编码
Positional Encoding]
    C --> D[多头注意力
Multi-Head Attention]
    D --> E[前馈网络
Feed Forward]
    E --> F[输出]

注意力机制

自注意力机制是 Transformer 的核心创新：

sequenceDiagram
    participant Q as Query
    participant K as Key
    participant V as Value
    
    Q->>K: 计算相似度
    K-->>Q: 返回权重
    Q->>V: 加权求和
    V-->>Q: 输出结果
