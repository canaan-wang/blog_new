Redis 事务

一、事务简介

Redis 事务将多个命令打包执行，保证命令按顺序执行且不被其他命令打断。但不支持回滚。

二、事务命令

MULTI       # 开启事务
EXEC        # 执行事务
DISCARD     # 取消事务
WATCH key   # 监视键（乐观锁）
UNWATCH     # 取消监视

三、基本用法

127.0.0.1:6379> MULTI
OK
127.0.0.1:6379> SET balance 100
QUEUED
127.0.0.1:6379> INCR balance
QUEUED
127.0.0.1:6379> GET balance
QUEUED
127.0.0.1:6379> EXEC
1) OK
2) (integer) 101
3) "101"

四、事务执行流程

graph TD
    A[MULTI] --> B[命令入队]
    B --> C{EXEC?}
    C -->|是| D[执行所有命令]
    C -->|否| E{DISCARD?}
    E -->|是| F[清空队列]
    E -->|否| B

五、Java 实现

// Jedis 事务
Jedis jedis = new Jedis("localhost");
Transaction tx = jedis.multi();
tx.set("key1", "value1");
tx.incr("counter");
tx.lpush("list", "item");
