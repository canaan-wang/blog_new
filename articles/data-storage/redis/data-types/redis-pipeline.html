Redis Pipeline

一、Pipeline 简介

Pipeline 通过一次性发送多个命令并批量接收结果，减少网络往返时间（RTT），显著提升批量操作性能。

二、性能对比

graph LR
    A[客户端] -->|CMD1| B[Redis]
    B -->|R1| A
    A -->|CMD2| B
    B -->|R2| A
    A -->|CMD3| B
    B -->|R3| A
    
    C[客户端] -->|CMD1| D[Redis]
    C -->|CMD2| D
    C -->|CMD3| D
    D -->|R1 R2 R3| C
方式1000 条命令耗时
普通1000 × RTT
Pipeline1 × RTT
