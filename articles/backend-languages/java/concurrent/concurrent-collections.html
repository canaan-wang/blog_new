三、BlockingQueue

队列特点

LinkedBlockingQueue可选有界链表
PriorityBlockingQueue优先级
SynchronousQueue直接传递
DelayQueue延迟获取

四、生产者消费者示例

BlockingQueue queue = new LinkedBlockingQueue<>(100);

// 生产者
queue.put(task);      // 阻塞
queue.offer(task, 1, TimeUnit.SECONDS);  // 超时

// 消费者
Task task = queue.take();    // 阻塞
Task task = queue.poll(1, TimeUnit.SECONDS);  // 超时
