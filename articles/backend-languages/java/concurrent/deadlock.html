Java 死锁

一、死锁条件

Coffman 条件（必须同时满足）：

  1. 互斥：资源一次只能被一个进程占用
  2. 占有等待：持有资源的同时等待其他资源
  3. 不可抢占：已分配的资源不能被强制剥夺
  4. 循环等待：进程间形成循环等待链

二、死锁示例

public class DeadlockDemo {
    private static final Object lockA = new Object();
    private static final Object lockB = new Object();
    
    public static void main(String[] args) {
        Thread t1 = new Thread(() -> {
            synchronized (lockA) {
                System.out.println("T1: Holding lock A");
                sleep(100);
                synchronized (lockB) {  // 等待 lockB
                    System.out.println("T1: Holding lock B");
                }
            }
        });
        
        Thread t2 = new Thread(() -> {
            synchronized (lockB) {
                System.out.println("T2: Holding lock B");
                sleep(100);
                synchronized (lockA) {  // 等待 lockA
                    System.out.println("T2: Holding lock A");
                }
            }
        });
        
        t1.start();
        t2.start();
    }
}
