Java CountDownLatch

一、简介

CountDownLatch 是一个同步工具类，用于一个或多个线程等待其他线程完成操作。内部维护一个计数器，减到 0 时等待线程被唤醒。

二、基本用法

import java.util.concurrent.CountDownLatch;

public class CountDownLatchDemo {
    public static void main(String[] args) throws InterruptedException {
        int workerCount = 3;
        CountDownLatch latch = new CountDownLatch(workerCount);
        
        for (int i = 0; i < workerCount; i++) {
            final int id = i;
            new Thread(() -> {
                System.out.println("Worker " + id + " started");
                try {
                    Thread.sleep(1000);
                } catch (InterruptedException e) {
                    Thread.currentThread().interrupt();
                }
                System.out.println("Worker " + id + " finished");
                latch.countDown();  // 计数器减 1
            }).start();
        }
        
        System.out.println("Main thread waiting...");
        latch.await();  // 等待计数器归零
        System.out.println("All workers finished!");
    }
}

三、核心方法

方法说明
CountDownLatch(int count)构造方法，指定初始计数
void await()阻塞等待，直到计数为 0
boolean await(long timeout, TimeUnit unit)超时等待
void countDown()计数器减 1
long getCount()获取当前计数

四、应用场景

1. 多任务并行后汇总

public class ParallelDataLoader {
    public Map loadData() throws InterruptedException {
        CountDownLatch latch = new CountDownLatch(3);
        Map result = new ConcurrentHashMap<>();
        
        // 并行加载用户数据
        executor.execute(() -> {
            result.put("user", loadUser());
            latch.countDown();
        });
        
        // 并行加载订单数据
        executor.execute(() -> {
            result.put("orders", loadOrders());
            latch.countDown();
        });
        
        // 并行加载配置数据
        executor.execute(() -> {
            result.put("config", loadConfig());
            latch.countDown();
        });
        
        latch.await();  // 等待全部完成
        return result;
    }
}

2. 服务启动检查

public class ServiceStarter {
    private final CountDownLatch startupLatch = new CountDownLatch(1);
    
    public void start() {
        new Thread(() -> {
            // 初始化各种资源
            initDatabase();
            initCache();
            initMQ();
            startupLatch.countDown();  // 启动完成
        }).start();
    }
    
    public void waitForStartup() throws InterruptedException {
        startupLatch.await();  // 等待启动完成
    }
}

3. 压力测试等待

public class LoadTest {
    public void runTest(int threadCount) throws InterruptedException {
        CountDownLatch readyLatch = new CountDownLatch(threadCount);
        CountDownLatch startLatch = new CountDownLatch(1);
        CountDownLatch doneLatch = new CountDownLatch(threadCount);
        
        for (int i = 0; i < threadCount; i++) {
            new Thread(() -> {
                readyLatch.countDown();  // 报告准备就绪
                try {
                    startLatch.await();  // 等待发令枪
                    // 执行业务测试
                    doWork();
                } catch (InterruptedException e) {
                    Thread.currentThread().interrupt();
                } finally {
                    doneLatch.countDown();
                }
            }).start();
        }
        
        readyLatch.await();     // 等待所有线程就绪
        long start = System.currentTimeMillis();
        startLatch.countDown(); // 发令：同时开始！
        doneLatch.await();      // 等待所有线程完成
        System.out.println("Total time: " + (System.currentTimeMillis() - start));
    }
}

五、注意事项

  • 计数器不能重置，只能用一次
  • countDown() 不会阻塞，await() 才会
  • 计数为 0 后再调用 await() 立即返回
  • 适合一个/多个线程等一个/多个线程的场景
