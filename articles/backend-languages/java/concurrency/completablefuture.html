Java CompletableFuture

一、简介

CompletableFuture 是 Java 8 引入的异步编程工具，支持函数式组合、链式调用，简化了异步任务处理。

二、创建异步任务

import java.util.concurrent.CompletableFuture;

// 无返回值
CompletableFuture future = CompletableFuture.runAsync(() -> {
    System.out.println("Running in " + Thread.currentThread().getName());
});

// 有返回值
CompletableFuture future = CompletableFuture.supplyAsync(() -> {
    return "Hello";
});

// 使用自定义线程池
ExecutorService executor = Executors.newFixedThreadPool(4);
CompletableFuture future = CompletableFuture.supplyAsync(() -> {
    return "World";
}, executor);

三、链式操作

// thenApply: 转换结果
CompletableFuture future = CompletableFuture.supplyAsync(() -> "hello")
    .thenApply(s -> s.toUpperCase())
    .thenApply(s -> s + " WORLD");
// 结果: HELLO WORLD

// thenAccept: 消费结果（无返回值）
CompletableFuture.supplyAsync(() -> "data")
    .thenAccept(System.out::println);

// thenRun: 不依赖结果，执行动作
CompletableFuture.supplyAsync(() -> "done")
    .thenRun(() -> System.out.println("Finished"));

四、组合多个 Future

// thenCompose: 扁平化（避免嵌套 Future）
CompletableFuture result = getUserId()
    .thenCompose(id -> getUserName(id));

// thenCombine: 合并两个独立 Future
CompletableFuture result = getUser()
    .thenCombine(getOrder(), (user, order) -> user + ":" + order);

// allOf: 等待所有完成
CompletableFuture all = CompletableFuture.allOf(future1, future2, future3);
