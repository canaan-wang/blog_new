Java 反射机制

一、反射简介

反射允许程序在运行时检查类、方法、字段等信息，并动态调用方法、创建实例。

二、获取 Class 对象

// 1. 类字面量
Class clazz = User.class;

// 2. 对象的 getClass()
User user = new User();
Class clazz = user.getClass();

// 3. Class.forName()
Class clazz = Class.forName("com.example.User");

// 4. 类加载器
Class clazz = ClassLoader.getSystemClassLoader()
    .loadClass("com.example.User");

三、创建实例

// 无参构造
User user = User.class.newInstance();  // 已废弃
User user = User.class.getDeclaredConstructor().newInstance();

// 有参构造
Constructor constructor = User.class.getConstructor(String.class, int.class);
User user = constructor.newInstance("Alice", 25);

四、访问字段

Class clazz = User.class;

// 获取字段
Field nameField = clazz.getDeclaredField("name");

// 访问私有字段
nameField.setAccessible(true);

// 读写
User user = new User();
nameField.set(user, "Bob");
String name = (String) nameField.get(user);

五、调用方法

Class clazz = User.class;

// 获取方法
Method setNameMethod = clazz.getMethod("setName", String.class);
Method getNameMethod = clazz.getMethod("getName");

// 调用
User user = new User();
setNameMethod.invoke(user, "Charlie");
String name = (String) getNameMethod.invoke(user);

六、获取类信息

Class clazz = User.class;

// 基本信息
String name = clazz.getName();          // 全限定名
String simpleName = clazz.getSimpleName();  // 简单名
Package pkg = clazz.getPackage();       // 包
Class superClass = clazz.getSuperclass();  // 父类
Class[] interfaces = clazz.getInterfaces(); // 接口

// 成员
Field[] fields = clazz.getDeclaredFields();     // 所有字段
Field[] publicFields = clazz.getFields();       // public 字段
Method[] methods = clazz.getDeclaredMethods();  // 所有方法
Constructor[] constructors = clazz.getDeclaredConstructors();

七、注解处理

// 获取类注解
MyAnnotation ann = clazz.getAnnotation(MyAnnotation.class);

// 获取方法注解
Method method = clazz.getMethod("doSomething");
MyAnnotation ann = method.getAnnotation(MyAnnotation.class);

// 获取字段注解
Field field = clazz.getDeclaredField("value");
MyAnnotation ann = field.getAnnotation(MyAnnotation.class);

// 判断是否有注解
boolean hasAnn = clazz.isAnnotationPresent(MyAnnotation.class);

八、反射应用示例

通用对象拷贝

public static void copyProperties(Object source, Object target) {
    Class sourceClass = source.getClass();
    Class targetClass = target.getClass();
    
    for (Field sourceField : sourceClass.getDeclaredFields()) {
        try {
            Field targetField = targetClass.getDeclaredField(sourceField.getName());
            sourceField.setAccessible(true);
            targetField.setAccessible(true);
            targetField.set(target, sourceField.get(source));
        } catch (NoSuchFieldException | IllegalAccessException e) {
            // 忽略
        }
    }
}

简单的依赖注入

public class SimpleContainer {
    public static void inject(Object obj) throws Exception {
        Class clazz = obj.getClass();
        for (Field field : clazz.getDeclaredFields()) {
            Autowired ann = field.getAnnotation(Autowired.class);
            if (ann != null) {
                field.setAccessible(true);
                Class fieldType = field.getType();
                Object instance = fieldType.getDeclaredConstructor().newInstance();
                field.set(obj, instance);
            }
        }
    }
}

九、反射优化

// Method/Field 缓存
private static final Map methodCache = new ConcurrentHashMap<>();

public static Method getCachedMethod(Class clazz, String name, Class... params) {
    String key = clazz.getName() + "." + name;
    return methodCache.computeIfAbsent(key, k -> {
        try {
            Method method = clazz.getMethod(name, params);
            method.setAccessible(true);
            return method;
        } catch (NoSuchMethodException e) {
            throw new RuntimeException(e);
        }
    });
}

十、反射注意事项

  • 性能较低（可用 MethodHandle 优化）
  • 破坏封装（setAccessible）
  • 编译时检查失效
  • Java 9+ 模块化可能受限
