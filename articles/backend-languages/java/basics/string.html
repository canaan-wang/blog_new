Java String

一、String 核心特性

String 是 Java 中使用最频繁的类，具有以下核心特性：

  • 不可变性：字符串一旦创建不可修改，修改会创建新对象
  • 常量池：字面量字符串存储在常量池，复用相同字符串
  • final 类：不可被继承，保证不可变性

二、创建方式对比

方式示例存储位置特点
字面量String s = "abc";字符串常量池复用，高效
newString s = new String("abc");堆内存每次新建对象
String s1 = "hello"; // 常量池
String s2 = "hello"; // 复用常量池对象
String s3 = new String("hello"); // 堆内存新对象

System.out.println(s1 == s2); // true，同一对象
System.out.println(s1 == s3); // false，不同对象

三、字符串拼接

3.1 + 运算符

String s = "a" + "b"; // 编译优化为 "ab"

// 循环中避免使用 +，每次都会创建新对象
String result = "";
for (String s : list) {
    result += s; // 每次创建 StringBuilder + 新 String
}

3.2 StringBuilder（推荐）

// 单线程使用 StringBuilder
StringBuilder sb = new StringBuilder();
for (String s : list) {
    sb.append(s);
}
String result = sb.toString();

// 多线程使用 StringBuffer（线程安全，性能略低）
StringBuffer sbf = new StringBuffer();
sbf.append("thread-safe");

五、比较方式

String s1 = "abc";
String s2 = new String("abc");

// == 比较引用地址
s1 == s2; // false

// equals 比较内容
s1.equals(s2); // true

// 常量比较，避免 NPE
"abc".equals(s1); // 推荐

// compareTo 比较大小
"a".compareTo("b"); // 负数，小于

六、String.intern()

将字符串放入常量池，返回常量池中的引用：

String s1 = new String("hello").intern();
String s2 = "hello";
System.out.println(s1 == s2); // true

七、最佳实践

  • 优先使用字面量创建字符串
  • 循环拼接用 StringBuilder，不用 +
  • 比较内容用 equals，不用 ==
  • 判断空串先判 null 再判空
  • Java 11+ 优先使用 strip() 而非 trim()
