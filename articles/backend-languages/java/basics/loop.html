Java 循环与遍历

一、传统循环方式

for 循环

// 基础 for
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// 倒序遍历（删除元素时常用）
for (int i = list.size() - 1; i >= 0; i--) {
    if (needRemove(list.get(i))) {
        list.remove(i);
    }
}

while / do-while

// while
while (condition) {
    // 可能一次都不执行
}

// do-while
do {
    // 至少执行一次
} while (condition);

二、增强 for 循环（foreach）

List list = Arrays.asList("a", "b", "c");

// 遍历元素
for (String item : list) {
    System.out.println(item);
}

// 遍历数组
int[] arr = {1, 2, 3};
for (int num : arr) {
    System.out.println(num);
}

// 注意：遍历时不能修改集合（会抛 ConcurrentModificationException）
for (String item : list) {
    // list.remove(item); // 错误！
}

三、Iterator 迭代器

// 基础用法
Iterator it = list.iterator();
while (it.hasNext()) {
    String item = it.next();
    if (needRemove(item)) {
        it.remove(); // 安全删除
    }
}

// ListIterator（支持双向遍历）
ListIterator lit = list.listIterator();
while (lit.hasNext()) {
    String item = lit.next();
    lit.set("new_" + item); // 修改元素
}
while (lit.hasPrevious()) {
    String item = lit.previous(); // 反向遍历
}

四、Stream API（Java 8+）

List numbers = Arrays.asList(1, 2, 3, 4, 5);

// 过滤
numbers.stream()
    .filter(n -> n > 2)
    .forEach(System.out::println);

// 映射
numbers.stream()
    .map(n -> n * n)
    .collect(Collectors.toList());

// 排序
numbers.stream()
    .sorted(Comparator.reverseOrder())
    .collect(Collectors.toList());

// 聚合
int sum = numbers.stream().mapToInt(Integer::intValue).sum();
Optional max = numbers.stream().max(Integer::compare);

五、遍历 Map

Map map = new HashMap<>();

// 方式1：entrySet
for (Map.Entry entry : map.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}

// 方式2：keySet
for (String key : map.keySet()) {
    System.out.println(key + " = " + map.get(key));
}

// 方式3：Java 8 forEach
map.forEach((k, v) -> System.out.println(k + " = " + v));

// 方式4：Stream
map.entrySet().stream()
    .filter(e -> e.getValue() > 10)
    .forEach(e -> System.out.println(e.getKey()));

六、循环控制

// break：跳出整个循环
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
}

// continue：跳过当前迭代
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    System.out.println(i); // 只打印奇数
}

// 带标签的 break（跳出多层循环）
outer:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (j == 1) break outer;
    }
}

七、最佳实践

  • 简单遍历用增强 for，需索引用传统 for
  • 遍历删除必须用 Iterator.remove()
  • 数据处理优先用 Stream API（简洁、易并行）
  • 大数据量避免频繁 Stream 装箱（用 mapToInt 等）
  • 嵌套循环考虑性能，必要时用标签 break
