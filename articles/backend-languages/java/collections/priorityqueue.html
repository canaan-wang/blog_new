Java PriorityQueue

一、简介

PriorityQueue 是基于堆实现的优先队列，保证队首是最小（或最大）元素。

二、类继承关系

graph TD
    Queue --> PriorityQueue
    AbstractQueue --> PriorityQueue

三、基本用法

import java.util.PriorityQueue;

// 自然排序（最小堆）
PriorityQueue pq = new PriorityQueue<>();
pq.offer(5);
pq.offer(1);
pq.offer(3);
System.out.println(pq.poll());  // 1
System.out.println(pq.poll());  // 3

// 自定义比较器（最大堆）
PriorityQueue maxHeap = new PriorityQueue<>((a, b) -> b - a);
maxHeap.offer(1);
maxHeap.offer(5);
maxHeap.offer(3);
System.out.println(maxHeap.poll());  // 5

四、核心方法

方法说明失败时
offer(E)添加元素返回 false
add(E)添加元素抛出异常
poll()移除并返回队首返回 null
remove()移除并返回队首抛出异常
peek()查看队首返回 null
element()查看队首抛出异常

五、自定义对象

class Task implements Comparable {
    int priority;
    String name;
    
    public Task(int priority, String name) {
        this.priority = priority;
        this.name = name;
    }
    
    @Override
    public int compareTo(Task other) {
        return this.priority - other.priority;
    }
}

PriorityQueue taskQueue = new PriorityQueue<>();
taskQueue.offer(new Task(3, "Low"));
taskQueue.offer(new Task(1, "High"));
taskQueue.offer(new Task(2, "Medium"));

Task task = taskQueue.poll();
System.out.println(task.name);  // High

六、应用场景

Top K 问题

// 找数组中最大的 K 个数
public List topK(int[] nums, int k) {
    PriorityQueue minHeap = new PriorityQueue<>(k);
    
    for (int num : nums) {
        if (minHeap.size() < k) {
            minHeap.offer(num);
        } else if (num > minHeap.peek()) {
            minHeap.poll();
            minHeap.offer(num);
        }
    }
    
    return new ArrayList<>(minHeap);
}

合并 K 个有序数组

public List mergeKLists(List> lists) {
    PriorityQueue pq = new PriorityQueue<>(Comparator.comparingInt(a -> a[0]));
    List result = new ArrayList<>();
    
    // 初始化：每个数组的第一个元素
    for (int i = 0; i < lists.size(); i++) {
        if (!lists.get(i).isEmpty()) {
            pq.offer(new int[]{lists.get(i).get(0), i, 0});
        }
    }
    
    while (!pq.isEmpty()) {
        int[] curr = pq.poll();
        result.add(curr[0]);
        
        int listIdx = curr[1], elemIdx = curr[2];
        if (elemIdx + 1 < lists.get(listIdx).size()) {
            pq.offer(new int[]{
                lists.get(listIdx).get(elemIdx + 1), 
                listIdx, elemIdx + 1
            });
        }
    }
    
    return result;
}

任务调度

// 延迟任务队列
PriorityQueue delayQueue = new PriorityQueue<>(
    Comparator.comparingLong(t -> t.executeTime)
);

delayQueue.offer(new DelayedTask("Task1", System.currentTimeMillis() + 5000));
delayQueue.offer(new DelayedTask("Task2", System.currentTimeMillis() + 3000));

// 检查并执行到期任务
while (!delayQueue.isEmpty() && 
       delayQueue.peek().executeTime <= System.currentTimeMillis()) {
    DelayedTask task = delayQueue.poll();
    task.execute();
}

七、PriorityQueue vs TreeSet

特性PriorityQueueTreeSet
重复元素允许不允许
遍历顺序无序有序
取极值O(1) 取队首O(1) first/last
查找任意元素O(n)O(log n)
适用场景只需极值需要有序集合

八、注意事项

  • 非线程安全，多线程需用 PriorityBlockingQueue
  • 不支持 null 元素
  • 迭代器不保证有序
  • 优先用 offer/poll/peek 而非 add/remove/element
