Java TreeMap

一、简介

TreeMap 是基于红黑树实现的有序 Map，按键的自然顺序或自定义比较器排序。

二、类继承关系

graph TD
    Map --> SortedMap
    SortedMap --> NavigableMap
    NavigableMap --> TreeMap
    AbstractMap --> TreeMap

三、基本用法

import java.util.TreeMap;

// 自然排序（Key 需实现 Comparable）
TreeMap map = new TreeMap<>();
map.put(3, "Three");
map.put(1, "One");
map.put(2, "Two");
System.out.println(map);  // {1=One, 2=Two, 3=Three}

// 自定义比较器（降序）
TreeMap descMap = new TreeMap<>((a, b) -> b - a);
descMap.put(1, "One");
descMap.put(3, "Three");
descMap.put(2, "Two");
System.out.println(descMap);  // {3=Three, 2=Two, 1=One}

四、核心方法

方法说明
firstKey() / lastKey()最小/最大键
firstEntry() / lastEntry()最小/最大键值对
ceilingKey(K)大于等于给定键的最小键
floorKey(K)小于等于给定键的最大键
higherKey(K)严格大于给定键的最小键
lowerKey(K)严格小于给定键的最大键
pollFirstEntry() / pollLastEntry()删除并返回最小/最大键值对
subMap(K from, K to)返回子 Map[from, to)
headMap(K)返回小于给定键的子 Map
tailMap(K)返回大于等于给定键的子 Map

五、范围查询示例

TreeMap scores = new TreeMap<>();
scores.put(85, "B");
scores.put(90, "A");
scores.put(78, "C");
scores.put(92, "A+");

// 查找及格的成绩
SortedMap passed = scores.tailMap(60);
System.out.println(passed);  // {78=C, 85=B, 90=A, 92=A+}

// 查找 80-90 分的
SortedMap range = scores.subMap(80, 91);
System.out.println(range);  // {85=B, 90=A}

// 查找刚好大于等于 88 的
Integer key = scores.ceilingKey(88);
System.out.println(key);  // 90

六、红黑树特性

TreeMap 基于红黑树，保证：

  • 查找、插入、删除：O(log n)
  • 键有序存储

七、TreeMap vs HashMap

特性TreeMapHashMap
底层结构红黑树哈希表
顺序有序无序
时间复杂度O(log n)O(1)
Key 要求需 Comparable 或 Comparator需实现 hashCode/equals
null Key不支持支持一个

八、应用场景

// 排行榜（分数 -> 用户）
TreeMap> leaderboard = new TreeMap<>(Comparator.reverseOrder());

// 时间段查询（时间戳 -> 数据）
TreeMap events = new TreeMap<>();
List recent = new ArrayList<>(events.tailMap(System.currentTimeMillis() - 86400000).values());
