Java Stream API

一、Stream 简介

Stream 是 Java 8 引入的函数式编程特性，用于对集合进行声明式数据处理，支持链式操作和并行执行。

二、创建 Stream

// 从集合创建
List list = Arrays.asList(1, 2, 3);
Stream stream = list.stream();

// 直接创建
Stream stream = Stream.of(1, 2, 3);

// 无限流
Stream infinite = Stream.iterate(0, n -> n + 2).limit(10);

三、中间操作

操作说明
filter(Predicate)过滤元素
map(Function)转换元素
flatMap(Function)扁平化映射
distinct()去重
sorted()排序
limit(n)截取前 n 个
skip(n)跳过前 n 个

四、终结操作

操作说明
collect(Collector)收集结果
forEach(Consumer)遍历
reduce(BinaryOperator)归约
count()计数
anyMatch/allMatch匹配检查
findFirst/findAny查找元素

五、Collectors 工具

// 转 List
List list = stream.collect(Collectors.toList());

// 转 Map
Map map = users.stream()
    .collect(Collectors.toMap(User::getId, User::getName));

// 分组
Map> byCity = users.stream()
    .collect(Collectors.groupingBy(User::getCity));

// 统计
IntSummaryStatistics stats = numbers.stream()
    .collect(Collectors.summarizingInt(Integer::intValue));

六、最佳实践

  • Stream 只能消费一次
  • 优先使用方法引用
  • 避免在 Stream 中修改外部变量
  • 谨慎使用并行流
