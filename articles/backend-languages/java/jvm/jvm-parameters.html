JVM 参数配置

一、标准参数

java -version
java -help
java -cp classpath MainClass
java -jar app.jar

二、堆内存参数

参数说明示例
-Xms初始堆大小-Xms512m
-Xmx最大堆大小-Xmx2g
-Xmn新生代大小-Xmn256m
-XX:NewRatio老年代/新生代比例-XX:NewRatio=2
-XX:SurvivorRatioEden/Survivor 比例-XX:SurvivorRatio=8
# 推荐设置：Xms = Xmx，避免动态扩缩容
java -Xms2g -Xmx2g -Xmn512m -jar app.jar

三、元空间参数

参数说明
-XX:MetaspaceSize初始元空间大小（JDK8+）
-XX:MaxMetaspaceSize最大元空间大小
-XX:PermSize初始永久代（JDK7）
-XX:MaxPermSize最大永久代（JDK7）
java -XX:MetaspaceSize=128m -XX:MaxMetaspaceSize=256m

四、GC 参数

选择垃圾收集器

参数收集器适用场景
-XX:+UseSerialGCSerial + Serial Old单核/小内存
-XX:+UseParNewGCParNew + CMS多核
-XX:+UseConcMarkSweepGCParNew + CMS低延迟
-XX:+UseParallelGCParallel Scavenge + Parallel Old高吞吐（默认）
-XX:+UseG1GCG1大堆/低延迟（JDK9+默认）
-XX:+UseZGCZGC超低延迟（JDK15+）

GC 调优参数

# Parallel GC 调优
-XX:MaxGCPauseMillis=200    # 最大停顿时间目标
-XX:GCTimeRatio=99          # GC 时间占比 1%

# CMS 调优
-XX:CMSInitiatingOccupancyFraction=70  # 老年代70%时触发CMS
-XX:+UseCMSCompactAtFullCollection     # Full GC 时压缩
-XX:CMSFullGCsBeforeCompaction=0       # 每次 Full GC 都压缩

# G1 调优
-XX:MaxGCPauseMillis=200
-XX:G1HeapRegionSize=16m
-XX:InitiatingHeapOccupancyPercent=45

五、日志参数

# JDK 8
-XX:+PrintGCDetails
-XX:+PrintGCDateStamps
-Xloggc:/var/log/gc.log
-XX:+UseGCLogFileRotation
-XX:NumberOfGCLogFiles=10
-XX:GCLogFileSize=100M

# JDK 9+
-Xlog:gc*:file=/var/log/gc.log:time,uptime,level,tags:filecount=10,filesize=100M

六、OOM 处理

# OOM 时自动生成堆转储
-XX:+HeapDumpOnOutOfMemoryError
-XX:HeapDumpPath=/var/log/heapdump.hprof

# OOM 时执行脚本
-XX:OnOutOfMemoryError="sh /cleanup.sh"

# 退出前打印类直方图
-XX:+PrintClassHistogramBeforeFullGC

七、常用配置组合

Web 应用（8G 内存）

java \
  -Xms4g -Xmx4g \
  -XX:MetaspaceSize=256m \
  -XX:MaxMetaspaceSize=512m \
  -XX:+UseG1GC \
  -XX:MaxGCPauseMillis=200 \
  -XX:+HeapDumpOnOutOfMemoryError \
  -XX:HeapDumpPath=/logs/heapdump.hprof \
  -Xlog:gc*:file=/logs/gc.log:time,uptime,level,tags \
  -jar app.jar

微服务（2G 内存）

java \
  -Xms1g -Xmx1g \
  -XX:+UseParallelGC \
  -XX:MaxGCPauseMillis=100 \
  -jar service.jar

八、查看当前参数

# 查看全部参数
java -XX:+PrintFlagsFinal -version | grep "= true"

# 查看修改过的参数
jinfo -flags PID
