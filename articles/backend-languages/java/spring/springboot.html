Spring Boot 入门

一、Spring Boot 是什么

Spring Boot 是 Spring 的扩展，提供自动配置、起步依赖和内嵌容器，快速构建生产级应用。

二、快速开始

1. 创建项目

六、自动配置原理

@SpringBootApplication = @Configuration + @EnableAutoConfiguration + @ComponentScan

graph TD
    A[启动应用] --> B[加载 spring.factories]
    B --> C[筛选条件配置类]
    C --> D{@Conditional 条件}
    D -->|匹配| E[应用自动配置]
    D -->|不匹配| F[跳过]
    E --> G[生成 Bean]

常用条件注解：

  • @ConditionalOnClass - 类存在
  • @ConditionalOnMissingBean - Bean 不存在
  • @ConditionalOnProperty - 配置属性
  • @ConditionalOnWebApplication - Web 应用

八、打包与运行

# 打包
mvn clean package

# 运行

# 指定配置
java -jar myapp.jar --server.port=8081 --spring.profiles.active=prod

九、最佳实践

  • 使用 Spring Initializr 创建项目
  • 优先用 application.yml
  • 敏感配置用环境变量或配置中心
  • 用 @ConfigurationProperties 组织配置
  • 启用 Actuator 监控应用健康
  • 用 Spring Boot DevTools 热部署
