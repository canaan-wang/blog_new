Spring Framework 概述

一、Spring 是什么

Spring 是一个轻量级的 Java 开发框架，核心特性是 IoC（控制反转） 和 AOP（面向切面编程），简化企业级应用开发。

二、核心特性

1. IoC 容器

控制反转：对象创建和依赖注入由容器管理，降低耦合。

// 传统方式
Service service = new Service();
service.setDao(new Dao());

// Spring IoC
@Service
public class Service {
    @Autowired
    private Dao dao;  // 自动注入
}

2. AOP 面向切面编程

@Aspect
@Component
public class LoggingAspect {
    @Before("execution(* com.service.*.*(..))")
    public void logBefore(JoinPoint jp) {
        System.out.println("Method called: " + jp.getSignature());
    }
}

3. 数据访问

  • JDBC 模板简化数据库操作
  • 统一事务管理
  • 集成 JPA、MyBatis

4. Spring MVC

@Controller
public class UserController {
    
    @GetMapping("/users/{id}")
    @ResponseBody
    public User getUser(@PathVariable Long id) {
        return userService.findById(id);
    }
}

四、Spring vs Spring Boot

特性SpringSpring Boot
配置大量 XML/注解配置自动配置，约定大于配置
依赖管理手动管理Starter 简化
部署外部容器内嵌容器，可执行 jar
监控需集成内置 Actuator
